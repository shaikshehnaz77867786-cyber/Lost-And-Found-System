Lost & Found System

A Case Study on a Simple Python-Based Solution

📌 Problem

Students and staff frequently lose items like phones, bags, and IDs. There is no proper system to track or recover them. Manual methods like paper logs are inefficient and unreliable.

💡 Our Solution

We developed a simple Python-based system that allows users to report lost and found items. The system stores data using lists and provides search functionality to find matches easily.

⚙️ Features

  • Report Lost Items
  • Report Found Items
  • View All Records
  • Search Items
  • Match Lost and Found Items

🧠 How It Works

Users enter item details through a menu-driven system. The program stores information in Python lists and compares entries to find possible matches.

🏗️ System Architecture

  • Input: User input via console
  • Processing: Python functions
  • Storage: Lists
  • Output: Console display

✅ Advantages

  • Easy to use
  • No database required
  • Fast and efficient
  • Improves item recovery

🌍 Conclusion

This project demonstrates that simple Python programs can solve real-world problems effectively. It can be extended into a full application with GUI or database integration.
